Transblogrify
=============

Transblogrify is a static site/blog generator for [node.js][1].

Requires
--------

   * [Node Markdown][2]
   * [YAML for CommonJS][3]
   * [Handlebars][4]

[1]: http://nodejs.org
[2]: http://github.com/andris9/node-markdown
[3]: http://github.com/visionmedia/js-yaml
[4]: http://github.com/wycats/handlebars.js
